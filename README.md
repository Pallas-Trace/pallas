# HTF

HTF provides an interface to write and read trace data.

## Building

```
cmake . -DCMAKE_INSTALL_PREFIX=$INSTALLATION_DIR
make install
```

## Using HTF

HTF implements a subset of the [OTF2](https://www.vi-hps.org/projects/score-p) API.
